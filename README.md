# Aiyl Bank (Test Task)

Бэкенд-сервис для имитации банковских переводов между счетами. Проект демонстрирует навыки построения отказоустойчивой бизнес-логики, обработки транзакций и документирования API.

## Ключевые особенности

* **Идемпотентность:** Реализована проверка по `idempotencyKey`. Повторные запросы с тем же ключом не создают новых транзакций, а возвращают результат существующей.
* **Транзакционный Аудит:** Система сохраняет информацию о каждой попытке перевода. В случае ошибки бизнес-логики статус меняется на `FAILED`, а причина (например, "Недостаточно средств") записывается в базу данных.
* **Валидация данных:** Использование `Jakarta Validation` и кастомных бизнес-проверок (запрет перевода самому себе, проверка активности счетов, проверка баланса).
* **Глобальная обработка ошибок:** Использование `@ControllerAdvice` для возврата понятных JSON-ответов вместо стандартных ошибок Spring.

## Технологический стек

* **Java 17**
* **Spring Boot 3.x**
* **Spring Data JPA** (PostgreSQL)
* **Lombok** (для сокращения шаблонного кода)
* **Swagger/OpenAPI** (для интерактивной документации)

* Запущенный **PostgreSQL**.
